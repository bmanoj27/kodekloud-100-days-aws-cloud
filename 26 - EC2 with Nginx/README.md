# EC2 Instance with Nginx Setup

## Task Overview

Create an EC2 instance with the following specifications:

### EC2 Instance
- **Instance Name:** `devops-ec2`
- **AMI:** Use any available **Ubuntu AMI**

### User Data Script
Configure the instance to run a user data script at launch that performs the following actions:
- Install the **Nginx** package
- Start the **Nginx** service

### Security Group
- Allow **HTTP traffic (port 80)** from the internet
